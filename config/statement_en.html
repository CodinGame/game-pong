The Goal

Send the ball behind the your opponent's paddle.

  Rules

The game is played on a map 1920 units wide and 1080 units high.

The ball has a radius of 20 units.

The paddles are 15 units wide and 150 units high.

The paddles can move up to 120 units per turn, vertically.

The ball's starting position is x=960, y=540.

The ball's starting velocity is vx=148, vy=132.

  Game Input

Input for one game turn
Line 1: 1 integer y for the current vertical position of your paddle.
Line 2: 1 integer oppY for the current vertical position of your opponent's paddle.
Line 3: 1 integer ballX for the current x coordinate of the ball.
Line 4: 1 integer ballY for the current y coordinate of the ball.
Line 5: 1 integer speedX for the current vx speed of the ball.
Line 6: 1 integer speedY for the current vy speed of the ball.
Output for one game turn
Line 1: 1 integer for the destination y position of your paddle.
Constraints
Response time for first turn ≤ 1000ms
Response time for one turn ≤ 100ms
